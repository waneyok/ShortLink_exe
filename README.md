# Short Link

## О приложении

**Short Link** — локальное приложение для создания коротких ссылок без подключения к внешним сервисам (БД, API, серверам). Всё хранится на вашем устройстве.

Предназначено для:
* быстрого сокращения длинных URL‑адресов;
* локального управления списком ссылок;

## Технические детали

* **Платформа**: Windows (WPF).
* **Фреймворк**: .NET Framework 4.8.
* **Интерфейс**: WPF (XAML + C#).
* **Зависимости**: отсутствуют (чистый .NET Framework).

## Установка и запуск

1. Скачайте архив `ShortLink.zip` из раздела Releases.
2. Распакуйте папку на ваш компьютер.
3. Запустите `ShortLink.exe`.
4. Приложение готово к работе — интернет не требуется.

## Использование

1. **Введите длинную ссылку** в поле «Исходная ссылка».
2. **Нажмите «Отправить»** — система сгенерирует короткий идентификатор.
3. **Скопируйте ссылку** кнопкой «Копировать» или вручную из поля вывода.
4. **Удаление** — используйте кнопку «Удалить» рядом с нужной ссылкой.

## Ограничения

* Нет синхронизации между устройствами.
* Данные удаляются при удалении папки с приложением.
* Нет истории ссылок
* Ссылки не работают, если закрыть приложение и заново использовать ссылку
* Не поддерживает кастомные короткие URL (генерируются автоматически).
